## Архитектура

`Feature Sliced Design`

Я понимаю, что эта архитектура очень избыточна для подобной величины проекта. Однако, я решил, что будет нелишним отобразить, как выглядит построение крупного приложения с самого нуля.

Это решение сильно облегчит дальнейшее (потенциальное) расширение проекта в неограниченном масштабе.

## Интересное

- добавил в `vite.config.js` и `tsconfig.json` возможность импорта с помощью `alias`. Это крайне удобно в проектах с подобной архитектурой;
- реализовал возможность добавления неограниченного количества цветовых тем с помощью `css variables`;
- добавил `prettier` и `stylelint`. Просто, чтобы радовать глаз ). Посчитал, что добавлять `eslint` не нужно, хотя можно было бы
